HOW TO CREATE VITE-VUE:

npm create vue@latest

cd %project-name%

npm install

git pull
	package.json > scripts > dev--open

npm run dev

**

NOTES:
- index.html is the loading point of root JS script (in case you rename it)
- vite.config.js > base: './' is the domain subdirectory (https://example.com/myapp/)
